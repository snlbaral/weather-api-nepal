# weather-api-nepal

Homepage:

		https://jcloudia.com/api


Endpoint URLs:
		
		https://jcloudia.com/api/weather

Usage: 

	https://jcloudia.com/api/weather{?OPTIONAL}
	
	{OPTIONAL}:
		'place': {PARAMETERS}
				'kathmandu',


Example:

	https://jcloudia.com/api/weather
	https://jcloudia.com/api/weather?place=kathmandu
